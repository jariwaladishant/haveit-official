# HAVE IT • Premium Food Experience 🍽️

![Have It Website Preview](images/have-it-back.png)

A stunning, modern "Coming Soon" website for a premium food brand, built with HTML5, CSS3, and JavaScript. Features a countdown timer, newsletter signup, image gallery, and PWA support.

## ✨ Features

### 🎨 **Design & UI**
- **Modern Dark Theme** with vibrant orange accents (`#ff8114`)
- **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- **Smooth Animations** - AOS (Animate on Scroll) library integration
- **Glass Morphism** - Beautiful backdrop blur effects
- **Custom Typography** - Luckiest Guy & Poppins fonts

### ⚡ **Functionality**
- **Live Countdown Timer** - Real-time launch countdown
- **Newsletter Signup** - Email collection with localStorage
- **Image Gallery** - Showcase your food products
- **Social Sharing** - WhatsApp, Twitter, Facebook share buttons
- **Dark/Light Mode** - Theme toggle with persistent storage

### 📱 **PWA Features**
- **Service Worker** - Offline caching
- **Manifest.json** - Install as a mobile app
- **Optimized Assets** - WebP image support with JPEG fallback
- **Fast Loading** - Lazy loading images

### 🔧 **Technical**
- **Modular Code** - Separate HTML, CSS, and JS files
- **SEO Optimized** - Meta tags, Open Graph, Twitter Cards
- **Accessibility** - ARIA labels, semantic HTML
- **Cross-browser** - Works on all modern browsers
